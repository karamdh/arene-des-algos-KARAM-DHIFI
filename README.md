# Arène des Algos — KARAM DHIFI

Projet fil rouge de la semaine ML (IPSSI).
Objectif : comparer plusieurs algorithmes de Machine Learning sur différents datasets et justifier le choix du meilleur.

## Programme

- **J1** : Pipeline supervisé + non-supervisé, première Arène
- **J2** : Nettoyage de données
- **J3** : Arsenal complet des algos
- **J4** : Réseaux de neurones, évaluation, déploiement
- **J5** : Projet groupe + soutenance
